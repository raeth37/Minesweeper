## Downloading Flutter
Follow the tutorial here: [installing Flutter for VSCode](https://docs.flutter.dev/install/with-vs-code).

## Running Minesweeper App
Navigate into UI folder to start running flutter app (built for Windows/Linux/MacOS)

1) Run `flutter clean` to clean workspace.
2) Run `flutter pub get` to download all dependencies.
3) Run `flutter run` to start building app.
    1) At this point, it will ask which device to run on, choose any of the OS listed above.
